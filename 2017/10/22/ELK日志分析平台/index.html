#第一部分 java环境搭建

#一、在centos 7系统安装java环境

###1.安装java1.0.8_151

进入到/usr/local/目录

cd /usr/local/

创建目录tools

mkdir -p tools、

进入到目录tools中

cd tools

下载java安装包

wget --no-cookies --header "Cookie: oraclelicense=accept-securebackup-cookie;" http://download.oracle.com/otn-pub/java/jdk/8u151-b12/e758a0de34e24606bca991d704f6dcbf/jdk-8u151-linux-x64.tar.gz

解压缩安装包

tar -xzf  jdk-8u151-linux-x64.tar.gz

解压完成

返回上一级目录，并创建/usr/local/jdk目录

cd ..
mkdir -p jdk

进入tools目录，将下载的jdk移动到jdk目录中

cd ../tools/
mv jdk1.8.0_151/ ../jdk/

###2、设置系统环境变量
打开 /etc/ 目录下的 profile 文件

vi /etc/profile

将如下代码追加到 profile 文件末尾：

 # idea - jdk8 settings start 2017-10-20

JAVA_HOME=/usr/local/jdk/jdk1.8.0_151
 JRE_HOME=$JAVA_HOME/jre
 PATH=$PATH:$JAVA_HOME/bin:$JRE_HOME/bin
 CLASSPATH=:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar:$JRE_HOME/lib/dt.jar
 export JAVA_HOME JRE_HOME PATH CLASSPATH

# idea - jdk8 settings end - 2017-10-20

设置完成

启用新的配置

source /etc/profile

测试完成！
